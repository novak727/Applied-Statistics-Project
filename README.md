# Applied Statistics I Project
This repository holds all files relating to the course project from my Applied Statistic I class. This includes the dataset in , the R file that contains all code that was written for analysis, the final PDF of the report, and the original LaTeX source file.

## Installation
You need [R](https://rstudio-education.github.io/hopr/starting.html) for any files in this repository, and [RStudio](https://posit.co/downloads/) is recommended for a friendlier interface.

### Option 1: using [git](https://git-scm.com/downloads)
1. Clone the repository
```
git clone https://github.com/novak727/Applied-Statistics.git
```
2. Navigate to the project directory
```
cd /Applied-Statistics/Lecture-Files
```
or
```
cd /Applied-Statistics/Homework-Files
```

### Option 2: without git
1. Download the project as a zip file
2. Extract the zip file

#### Running from the command line
1. Open a command line and go to the location of the files
```
cd /path/to/files
```
2. Run the following:
```
Rscript project-name.R
```
#### Running from RStudio
1. Click File -> Open File
2. Open project-name.R
3. Click Session -> Set Working Directory -> To Source File Location if the file is reading an external dataset
4. Click in the script, then click Ctrl+A (or Cmd+A on Mac)
5. Click the run button
6. Any graphics that are generated will be in the bottom right window. Use the left or right arrows to select a graph
